Gourmet is a social food review app for discovering and reviewing restaurants. Users can search for nearby restaurants, write       
  reviews with photos, and browse a personalized feed of reviews from others. An AI-powered recipe assistant (built on Claude) lets   
  users generate recipes inspired by their recent meals or dietary preferences. The app features a full authentication flow with      
  onboarding, user profiles, and a trending restaurants section surfaced from real review activity.
